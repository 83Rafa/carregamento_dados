Modelos de Fontes de Base de Dados

Como cientista de dados você não precisa decorar cada linha de código para realizar sua análise ou modelagem, só precisa saber onde as informações estão para quando precisar.

Embora com o tempo algumas linhas de código acabam se fixando na memória automaticamente, tenho sempre uma colinha de fácil acesso.

No meu caso, Python é de longe a ferramenta que mais uso, mas para algumas análises acabo utilizando o R também. Esse arquivo contém exemplos de carregamento de dados em Python e em R, espero que lhe seja útil!

  • Base de Dados Locais

    Começando pelo básico, os arquivos .csv não possuem muitos parâmetros, mas alguns podem ser fundamentais para carregar sua base, como separadores, idioma e índices. Outra coisa, no caso do Python é sempre bom se lembrar de em arquivos muito grandes que venham em formato compactado não há necessidade de descompactar na sua máquina, basta fazer a leitura em formato ‘zipado’: pd.read_csv(“arquivo.zip”) que o arquivo ficara salvo apenas na memória cache da ferramenta.

  • Base de Dados Através de APIs

    Nesse modelo, além do acesso via API você também vai notar que o DataFrame e a inserção dos dados foram gerados através de queries SQL, o que acho muito bacana nessas linguagens, porque a sintaxe é exatamente a mesma.

    Python

    import requests
import sqlite3

# URL da API do IBGE
url_ibge = "https://servicodados.ibge.gov.br/api/v3/agregados/7392/periodos/2014/variaveis/10484?localidades=N1[all]"

# Fazendo a requisição à API do IBGE
response = requests.get(url_ibge)
dados_ibge = response.json()

# Conectando ao banco de dados SQLite (o arquivo 'dados_ibge.db' será criado no diretório atual)
conn = sqlite3.connect('dados_ibge.db')
cursor = conn.cursor()

# Criando a tabela para armazenar os dados (se ainda não existir)
cursor.execute('''
CREATE TABLE IF NOT EXISTS dados_ibge (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    categoria TEXT,
    unidade TEXT,
    valor REAL
)
''')

# Inserindo os dados na tabela
for item in dados_ibge[0]['resultados'][0]['series']:
    categoria = item['categoria']['nome']
    unidade = item['unidade']['nome']
    valor = item['serie']['2014']

    cursor.execute('''
    INSERT INTO dados_ibge (categoria, unidade, valor) VALUES (?, ?, ?)
    ''', (categoria, unidade, valor))

# Salvando as alterações e fechando a conexão com o banco de dados
conn.commit()
conn.close()

print("Dados carregados com sucesso no banco de dados SQLite.")

    R

    # Bibliotecas necessárias
library(httr)
library(jsonlite)
library(DBI)
library(RSQLite)

# URL da API, neste exemplo, a do IBGE
url_ibge <- "https://servicodados.ibge.gov.br/api/v3/agregados/7392/periodos/2014/variaveis/10484?localidades=N1[all]"

# Fazendo a requisição à API
response <- GET(url_ibge)
dados_ibge <- content(response, "parsed")

# Conexão com o banco de dados SQLite (o arquivo .db será criado no diretório atual)
conn <- dbConnect(RSQLite::SQLite(), dbname = "dados_ibge.db")

# Criando a tabela para armazenar os dados (se ainda não existir)
dbExecute(conn, "CREATE TABLE IF NOT EXISTS dados_ibge (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    categoria TEXT,
    unidade TEXT,
    valor REAL
)")

# Inserindo os dados na tabela
for (item in dados_ibge[[1]]$resultados[[1]]$series) {
  categoria <- item$categoria$nome
  unidade <- item$unidade$nome
  valor <- item$serie$`2014`
  
  dbExecute(conn, "INSERT INTO dados_ibge (categoria, unidade, valor) VALUES (?, ?, ?)", params = list(categoria, unidade, valor))
}

# Salvando as alterações e fechando a conexão com o banco de dados
dbDisconnect(conn)

print("Dados carregados com sucesso no banco de dados SQLite.")
  • Base de dados em um Bucket S3 da Amazon

    Python

    import boto3
import pandas as pd
from io import StringIO

# Inicializando o cliente S3
s3_client = boto3.client('s3', aws_access_key_id='SUA_AWS_ACCESS_KEY_ID', 
                         aws_secret_access_key='SUA_AWS_SECRET_ACCESS_KEY')

# Nome do bucket e chave do arquivo no S3
bucket_name = 'seu-bucket-s3'
file_key = 'caminho/do/seu/arquivo.csv'

# Obtendo o objeto do S3
obj = s3_client.get_object(Bucket=bucket_name, Key=file_key)
data = obj['Body'].read().decode('utf-8')

# Carregando os dados em um DataFrame
df_s3 = pd.read_csv(StringIO(data))

df_s3.head()

  • Base de Dados da Azure

    Python

    from azure.storage.blob import BlobServiceClient, BlobClient, ContainerClient
import pandas as pd
from io import BytesIO

# Conexão com o Azure Blob Storage
connect_str = 'SUA_STRING_DE_CONEXÃO_AZURE'
blob_service_client = BlobServiceClient.from_connection_string(connect_str)

# Nome do container e nome do blob
container_name = 'seu-container'
blob_name = 'caminho/do/seu/arquivo.csv'

# Obtendo o blob
blob_client = blob_service_client.get_blob_client(container=container_name, 
                                                  blob=blob_name)
stream = blob_client.download_blob()
data = stream.readall()

# Carregando os dados em um DataFrame
df_azure = pd.read_csv(BytesIO(data))

df_azure.head()

    R
